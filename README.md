## 💡 리뷰하기 위해 알아야 할 것
1. 해당 repository를 fork 합니다. (Owner가 개인 계정으로 되어 있어야 합니다)
    - fork된 repo를 local에 clone해 사용합니다. 
2. 해당 repository에 브랜치를 생성합니다. 
    - 브랜치 이름은 각자의 이름을 소문자로 띄어쓰기나 특별문자 없이 작성합니다. (ex. hyeri)
3. 생성된 브랜치에 Commit 메시지 형식을 지켜 Commit 합니다.
